ABOUT CODEMEU

CODEMEU is a project created and managed by two students of IES San Clemente as ther final DAW project. CODEMEU was created with the purpose of making collaborative programming posible and accesible to all users, regardless of the platform they use. On the actual version of the project we provide collaborative editing, uploading and downloading of code, theme selector for the editor, code highlighting, autoclose tags, CSS and JS code completion and many more features that we will add and improve in the future.
